Elite Barber Studio - Full-Stack Booking System project

A complete full-stack booking management system built for a barbershop business. This professional-grade application features a modern, responsive public-facing website with an intelligent booking calendar that prevents double-bookings through real-time availability checking. The system includes secure token-based booking management allowing customers to view and cancel appointments, automated email confirmations with calendar integration, and a comprehensive admin panel with JWT authentication, revenue analytics, booking management with advanced search and filtering, customizable business hours with overtime buffer support, and mobile-optimized interfaces.
